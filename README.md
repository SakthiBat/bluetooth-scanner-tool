# Bluetooth Security Scanner Tool

## Description
A Bash-based Bluetooth security scanner that detects nearby and connected Bluetooth devices and identifies unknown devices to enhance wireless security.

## Features
- Scan nearby Bluetooth devices
- Show connected devices
- Detect unknown Bluetooth devices
- Maintain scan logs

## Tools Used
- Bash Scripting
- bluetoothctl
- hciconfig
- Kali Linux

## How to Run
```bash
chmod +x scanner.sh
sudo ./scanner.sh
